3. Description of the Application

Symptoms is an application designed for psychological exposure therapy using geolocation. Access to the app is granted via a code provided by a qualified therapist. The application offers the possibility to record audio and collect location data during therapeutic exposure.

4. Data We Collect

Access Code:
This code is assigned by the therapist through a separate web tool. We do not collect personal data (such as name, email, phone number); we only handle this code to manage user access.

Location Data:
We collect location data only during exposure sessions, provided the user has granted the relevant permissions (e.g., ACCESS_FINE_LOCATION, ACCESS_BACKGROUND_LOCATION on Android). The application ceases to collect location data once the exposure session ends.

Audio:
During an exposure session, users can record audio, which is stored in Base64 format in BigQuery. These recordings are collected for therapeutic purposes and are linked only to the access code provided by the therapist.

Device Data (minimal):
We may collect some technical information necessary to ensure the app functions correctly (e.g., app version, operating system). The app also requires storage permissions to save and send data when an internet connection is detected.

5. How We Use the Data

Therapeutic Purposes:
Location data and audio recordings are used so that the therapist can monitor exposures and provide better therapy follow-up. The therapist accesses this information through the associated web tool.

Storage and Synchronization:
Data is initially stored on the user’s device during sessions. Once the app detects an internet connection, data is uploaded to and stored in BigQuery. When the user logs out, local data is deleted from the device. When the user logs in, the app retrieves data stored in BigQuery.

Third-Party Services:
We use Firebase to manage user configurations (e.g., exposure locations) and BigQuery to store data. These providers may collect and process information according to their own privacy policies. We encourage you to review those to understand their data handling practices.

6. Permissions Requested

Our AndroidManifest.xml includes the following permissions:

  • android.permission.REQUEST_IGNORE_BATTERY_OPTIMIZATIONS
  • android.permission.ACCESS_FINE_LOCATION
  • android.permission.ACCESS_BACKGROUND_LOCATION
  • android.permission.READ_EXTERNAL_STORAGE
  • android.permission.WRITE_EXTERNAL_STORAGE
  • android.permission.INTERNET
  • android.permission.CALL_PHONE
  • android.permission.RECORD_AUDIO
  • android.permission.POST_NOTIFICATIONS

Purposes:
Location: to track geolocation during exposure sessions.
Storage: to temporarily store data (audio) and then upload to BigQuery.
Microphone: to record audio during exposure sessions.
Notifications: to send alerts or reminders related to therapy (if implemented).

7. Data Storage and Security

Local Storage: Data is temporarily stored on the device during exposure sessions.

Cloud Storage: Once the app detects an internet connection, data is synchronized and uploaded to BigQuery.

Security: We strive to protect your data via encryption methods and security protocols. However, information transmitted over the internet is not entirely secure, and we cannot guarantee absolute security.

8. User Rights

Although we do not collect personally identifiable information, users have the right to:

  • Access the data linked to their access code through their therapist.
  • Delete data: If the user decides to discontinue therapy, they can request the therapist to remove their records from BigQuery.
  • Opt out: Users may stop using the application at any time if they do not agree with location or audio recording practices.

9. Use by Minors

This application is intended for adult users. However, in exceptional cases it may be used by minors under the supervision of a therapist or legal guardian. We do not knowingly collect personal information from minors without consent.

10. Possible Data Collection from Third-Party Services

While using our Service, we do not require you to provide personally identifiable information. These services may collect data such as device information, app usage statistics, or other diagnostic information. By using our app, you acknowledge that these third parties may process your data according to their respective privacy policies.
