Políticas de Privacidad

Aplicación Móvil Symptoms (Última actualización: 17/01/2025)

1. Introducción

Bienvenido a Symptoms, una aplicación desarrollada por Grupo Geotec (“nosotros”, “nuestro”). Esta Política de Privacidad describe cómo recopilamos, usamos y protegemos la información del usuario (“usted”, “usuario”) dentro de la aplicación.
Al usar Symptoms, usted acepta los términos descritos en esta Política de Privacidad. Si no está de acuerdo con alguno de estos términos, por favor no use la aplicación.

3. Descripción de la Aplicación

Symptoms es una aplicación diseñada para terapia de exposición psicológica utilizando geolocalización. El acceso a la aplicación se otorga mediante un código proporcionado por un terapeuta calificado. La aplicación ofrece la posibilidad de grabar audio y recopilar datos de ubicación durante la exposición terapéutica.

4. Datos que Recopilamos

Código de Acceso:
Este código es asignado por el terapeuta a través de una herramienta web independiente. No recopilamos datos personales (como nombre, correo electrónico, número de teléfono); únicamente manejamos este código para gestionar el acceso del usuario.

Datos de Ubicación:
Recopilamos datos de ubicación solo durante las sesiones de exposición, siempre que el usuario haya otorgado los permisos correspondientes (por ejemplo, ACCESS_FINE_LOCATION, ACCESS_BACKGROUND_LOCATION en Android). La aplicación deja de recopilar datos de ubicación una vez finalizada la sesión de exposición.

Audio:
Durante una sesión de exposición, los usuarios pueden grabar audio, el cual se almacena en formato Base64 en BigQuery. Estas grabaciones se recopilan con fines terapéuticos y están vinculadas únicamente al código de acceso proporcionado por el terapeuta.

Datos del Dispositivo (mínimos):
Podemos recopilar cierta información técnica necesaria para asegurar el correcto funcionamiento de la aplicación (por ejemplo, versión de la app, sistema operativo). La aplicación también requiere permisos de almacenamiento para guardar y enviar datos cuando se detecte conexión a internet.

5. Cómo Usamos los Datos

Fines Terapéuticos:
Los datos de ubicación y las grabaciones de audio se utilizan para que el terapeuta pueda monitorear las exposiciones y brindar un mejor seguimiento terapéutico. El terapeuta accede a esta información a través de la herramienta web asociada.

Almacenamiento y Sincronización:
Los datos se almacenan inicialmente en el dispositivo del usuario durante las sesiones. Una vez que la aplicación detecta conexión a internet, los datos se suben y se guardan en BigQuery. Cuando el usuario cierra sesión, los datos locales se eliminan del dispositivo. Al iniciar sesión, la aplicación recupera los datos almacenados en BigQuery.

Servicios de Terceros:
Utilizamos Firebase para gestionar la configuración de los usuarios (por ejemplo, ubicaciones de exposición) y BigQuery para almacenar datos. Estos proveedores pueden recopilar y procesar información de acuerdo con sus propias políticas de privacidad. Le recomendamos revisarlas para comprender sus prácticas de manejo de datos.

6. Permisos Solicitados

Nuestro AndroidManifest.xml incluye los siguientes permisos:

  • android.permission.REQUEST_IGNORE_BATTERY_OPTIMIZATIONS
  • android.permission.ACCESS_FINE_LOCATION
  • android.permission.ACCESS_BACKGROUND_LOCATION
  • android.permission.READ_EXTERNAL_STORAGE
  • android.permission.WRITE_EXTERNAL_STORAGE
  • android.permission.INTERNET
  • android.permission.CALL_PHONE
  • android.permission.RECORD_AUDIO
  • android.permission.POST_NOTIFICATIONS

Propósitos:
Ubicación: para rastrear la geolocalización durante las sesiones de exposición.
Almacenamiento: para almacenar temporalmente datos (audio) y luego subirlos a BigQuery.
Micrófono: para grabar audio durante las sesiones de exposición.
Notificaciones: para enviar alertas o recordatorios relacionados con la terapia (si se implementa).

7. Almacenamiento y Seguridad de Datos

Almacenamiento Local: Los datos se almacenan temporalmente en el dispositivo durante las sesiones de exposición.

Almacenamiento en la Nube: Una vez que la aplicación detecta una conexión a internet, los datos se sincronizan y suben a BigQuery.

Seguridad: Nos esforzamos por proteger sus datos mediante métodos de cifrado y protocolos de seguridad. Sin embargo, la información transmitida por internet no es completamente segura y no podemos garantizar seguridad absoluta.

8. Derechos del Usuario

Aunque no recopilamos información personal identificable, los usuarios tienen derecho a:

  • Acceder a los datos vinculados a su código de acceso a través de su terapeuta.
  • Eliminar datos: Si el usuario decide interrumpir la terapia, puede solicitar al terapeuta que elimine sus registros de BigQuery.
  • Optar por no participar: El usuario puede dejar de usar la aplicación en cualquier momento si no está de acuerdo con las prácticas de grabación de audio o ubicación.

9. Uso por Menores

Esta aplicación está destinada a usuarios adultos. Sin embargo, en casos excepcionales puede ser utilizada por menores bajo la supervisión de un terapeuta o tutor legal. No recopilamos conscientemente información personal de menores sin consentimiento.

10. Posible Recolección de Datos de Servicios de Terceros

Mientras utiliza nuestro Servicio, no requerimos que proporcione información personalmente identificable. Sin embargo, la aplicación integra ciertos servicios de terceros que pueden recopilar información utilizada para identificarle. Recomendamos revisar la política de privacidad de cada servicio para entender sus prácticas de recolección y uso de datos.

Los proveedores de servicios de terceros utilizados por la aplicación incluyen:

  • Google Play services
  • Google Analytics for Firebase
  • Firebase Crashlytics
  • Mapbox
  • Couchbase
  • Akylas nativescript package

Estos servicios pueden recopilar datos como información del dispositivo, estadísticas de uso de la aplicación u otra información de diagnóstico. Al usar nuestra aplicación, usted reconoce que estos terceros pueden procesar sus datos de acuerdo con sus respectivas políticas de privacidad.

11. Cambios en Esta Política de Privacidad

Nos reservamos el derecho de actualizar o modificar esta Política de Privacidad en cualquier momento. Cuando se realicen cambios, notificaremos a los usuarios a través de:

  • Un aviso dentro de la aplicación.
  • Nuestro sitio web o la herramienta web asociada.
  • Un correo electrónico al terapeuta responsable, si corresponde.

La fecha de la última actualización se indica en la parte superior de este documento.
