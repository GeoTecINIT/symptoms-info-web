Plataforma SyMptoMS

SyMptoMS está formado por una aplicación móvil para los pacientes, una aplicación web para los terapeutas, servicios para guardar los datos y analizarlos, etc.

Nuestro objetivo

Nuestro objetivo es la creación de aplicaciones que usen las características avanzadas de los teléfonos móviles inteligentes, permitiendo configurar las aplicaciones según cada tipo de trastorno sin la necesidad de conocimientos técnicos.

Herramientas para el usuario final

Pacientes

Terapeutas

Aplicación Android

Tiene características configurables dependiendo del tratamiento. Monitorea el progreso del paciente a través de los sensores.

Aplicación Web de Gestión

Permite gestionar los centros, psicólogos y pacientes, así como definir la configuración de la aplicación móvil para cada paciente, ajustada a las necesidades de cada trastorno mental y tratamiento.

Aplicación Web de visualización

Permite hacer un seguimiento del progreso del paciente y analizar los datos reportados, tiene una vista de mapa para visualizar las exposiciones a unas localizaciones predefinidas. Permite visualizar las evaluaciones enviadas por cada paciente y las intervenciones recibidas.
